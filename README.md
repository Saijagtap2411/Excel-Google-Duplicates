# Excel Google Duplicates Checker

A PHP project to upload an Excel file, match numbers with a Google Sheet, and export the duplicates along with their count and associated dates in Excel format.

## Features

- Upload Excel files with numbers
- Check for duplicates against a Google Sheet
- Fetch duplicate count and dates from Column G
- Export the result to a downloadable Excel file

## Requirements

- PHP 7.4 or higher
- Composer
- Google Sheets API enabled
- PhpSpreadsheet library
